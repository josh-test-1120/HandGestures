Detection of Epileptic Seizures Through Motion Recognition Based on Machine Learning

This Capstone project focuses on developing a system to detect epileptic seizures using motion data from wearable IMU-equipped devices. It includes data collection, feature extraction, and predictive modeling—all visualized through a dynamic website.

Learn More View Research

How It Works

Arduino Hardware

A custom-built Arduino system with precision IMU sensors tracks hand motion and acceleration.

Motion Detection

Algorithms monitor hand movements in real-time to detect patterns related to seizure activity.

Data Analysis

Machine learning models process incoming data to predict and flag early signs of seizures.

Technical Implementation

  • ✔ MPU6050 Sensor Integration
    6-axis motion tracking for accurate acceleration and gyroscope readings.
  • ✔ Real-time Data Processing
    Arduino-based continuous data monitoring and signal processing.
  • ✔ Wireless Transmission
    Bluetooth connectivity for live data streaming and storage.
