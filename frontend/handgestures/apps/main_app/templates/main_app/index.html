Detection of Epileptic Seizures
Through Motion Recognition
Using Machine Learning

Our capstone project aims to identify seizure events using wearable IMU-based sensors. With real-time data collection and predictive modeling, our system offers a privacy-conscious and accessible method for non-invasive health monitoring.

Learn More View Research

How It Works

Arduino Hardware

A custom-built Arduino system with precision IMU sensors tracks hand motion and acceleration.

Motion Detection

Algorithms monitor hand movements in real-time to detect patterns related to seizure activity.

Data Analysis

Machine learning models process incoming data to predict and flag early signs of seizures.
