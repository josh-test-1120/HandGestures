Welcome to Our Group Project!

Project Overview:

This Capstone project focuses on developing a system to detect epileptic seizure by analyzing gesture data collected from wearable devices equipped with Inertial Measurement Units (IMUs). The project is emphasizing data collection, feature extraction, and modeling. The final deliverable for this phase includes a functional website that visually presents the collected IMU data, extracted features, and initial modeling results, along with a detailed report documenting the methodology and deployment process.
