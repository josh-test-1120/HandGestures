Early Detection of Epileptic Seizures
Through Motion Analysis

This Capstone project focuses on developing a system to detect epileptic seizure by analyzing gesture data collected from wearable devices equipped with Inertial Measurement Units (IMUs). The project emphasizes data collection, feature extraction, and modeling. The final deliverable includes a functional website that visually presents collected IMU data, extracted features, initial modeling results, and a detailed methodology report.
